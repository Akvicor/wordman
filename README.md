b3log-wordman
=============

沃德曼（Wordman）是一个移动端的背单词应用。

## 功能简介 ##
* 添加词库（同时可以学习多种词库）
* 每天都会生成当天的学习计划，词数默认为上一次的设置
* 开始学习一课时可以设置今天学习该课的词数，拼写一遍，正确的话到下一个词
* 艾宾浩斯记忆曲线：第 1 天、第 2 天、第 4 天、第 7 天和第 15天提醒，拼写后到下一个词，过完这课后重复错词，直到没有错词才进入下一课复习
